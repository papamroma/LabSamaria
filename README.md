# LabSamaria

A modern, AI-focused tech blog built with Next.js and a custom design system.

## Features

- **Modern UI**: Glassmorphism, custom gradients, and clean typography.
- **Theming**: Built-in Light and Dark mode toggle.
- **Content**: Articles, Categories, and Trending sections.
- **Responsive**: Optimized for all devices.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This project is ready to be deployed on Vercel or Netlify.

1.  Push this code to a GitHub repository.
2.  Import the repository into Vercel/Netlify.
3.  Deploy!

See [GIT_SETUP.md](./GIT_SETUP.md) for detailed instructions on how to connect this to GitHub.
